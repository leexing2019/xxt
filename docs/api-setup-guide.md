# API 配置指南

本文档指导管理员如何申请和配置第三方 API 以启用完整功能。

## 重要说明

**API 配置在后台管理系统中进行，不在前端 app 设置！**

- **后台管理地址**：`/admin` 目录部署的应用
- **配置位置**：后台管理系统 -> API 配置
- **数据存储**：API 密钥加密存储在 Supabase `api_config` 表中
- **前端读取**：前端 app 启动时自动从后端读取 API 配置（静默，无配置界面）

---

## 需要配置的 API

### 1. 百度 OCR API（必需）

**用途**：识别药品包装盒上的文字信息，自动填充药品信息。

**免费额度**：每月 500 次通用文字识别免费调用

**申请步骤**：

1. 访问百度智能云官网：https://cloud.baidu.com/
2. 注册/登录百度账号
3. 进入控制台 -> 人工智能 -> 文字识别 OCR
4. 创建应用，获取以下信息：
   - App ID
   - API Key
   - Secret Key
5. 开通"通用文字识别"服务

**配置位置**：后台管理系统 -> API 配置 -> 百度 OCR 配置

**文档链接**：https://cloud.baidu.com/doc/OCR/index.html

---

### 2. 百度语音 API（必需）

**用途**：
- 语音识别：说出药品名称自动搜索
- 语音播报：朗读用药提醒

**免费额度**：每月 10 万字符免费调用

**申请步骤**：

1. 访问百度智能云官网：https://cloud.baidu.com/
2. 注册/登录百度账号
3. 进入控制台 -> 人工智能 -> 语音技术
4. 创建应用，获取以下信息：
   - App ID
   - API Key
   - Secret Key
5. 开通"语音识别"和"语音合成"功能

**配置位置**：后台管理系统 -> API 配置 -> 百度语音配置

**文档链接**：https://cloud.baidu.com/doc/SPEECH/index.html

---

### 3. 药品数据库 API（可选）

**用途**：查询药品详细信息、禁忌症、药物相互作用等

**当前方案**：项目内置常用药品数据库，包含高血压、糖尿病等常见药品

**如需接入外部 API**，可配置：
- API 基础 URL
- API Key（如需要）

**配置位置**：后台管理系统 -> API 配置 -> 药品数据库配置

---

## 配置方式

### 通过后台管理系统配置（唯一方式）

1. 登录后台管理系统（`/admin` 目录部署的应用）
2. 进入"API 配置"页面
3. 展开对应的 API 区块
4. 填写 API 密钥
5. 点击"测试连接"验证
6. 点击"保存配置"

保存后，前端 app 会自动从后端获取配置，无需重启或刷新。

---

## 降级方案

如果管理员未在后台配置 API，前端将使用以下降级方案：

| 功能 | 降级方案 |
|------|----------|
| OCR 识别 | 返回模拟数据 |
| 语音识别 | 使用浏览器原生 Web Speech API（H5）|
| 语音播报 | 使用浏览器原生 Speech Synthesis API（H5）|
| 药品数据库 | 使用内置常用药品数据库 |

## 常见问题

### Q: 免费额度用完了怎么办？
A: 可以在百度智能云控制台付费升级，或者优化调用频率。

### Q: 可以在本地测试吗？
A: 可以，未配置 API 时使用模拟数据，不影响基础功能测试。

### Q: API 密钥安全吗？
A: API 密钥加密存储在 Supabase 数据库中，仅管理员可在后台查看和配置。

### Q: 支持其他 OCR 服务吗？
A: 当前支持百度 OCR，后续可扩展支持腾讯 OCR、阿里 OCR 等。
