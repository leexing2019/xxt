## 1. 概述

### 1.1 背景

原后台药品库管理有一个"批量导入常用药品"按钮，直接导入 50 种硬编码的常用药品。现需要改造为更灵活的 Excel 模板下载 + 用户上传的方式，允许管理员自定义批量添加药品。

### 1.2 目标

- 移除硬编码的批量导入功能
- 提供 Excel 模板下载，用户可离线填写
- 支持 Excel 文件上传和解析
- 提供预览弹窗，用户可当场修正错误数据
- 重复药品自动跳过并提示
- 导入完成后显示详细结果汇总

### 2.3 数据验证规则

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| 药品名称 | 必填，1-100 字 | "药品名称不能为空" |

**预设分类**：降压药、降糖药、降脂药、心血管药、胃药、止咳药、止痛药、维生素、钙片、抗生素、其他

### 2.4 重复药品处理

- **判断标准**：以"药品名称"字段为准，名称相同视为重复
- **处理方式**：跳过该行，不计入失败，单独统计为"跳过重复"
- **提示方式**：结果汇总中显示跳过数量，可展开查看明细

## 4. 技术架构

### 4.1 技术栈

- **xlsx (SheetJS)** - Excel 文件生成和解析
- **Element Plus** - UI 组件（el-dialog, el-table, el-upload）
- **Supabase Client** - 数据库操作

### 4.2 组件结构

```
Medications.vue
├── downloadTemplate()        - 生成并下载 Excel 模板
├── handleFileUpload()        - 解析 Excel 文件
├── validateRow()             - 验证单行数据
├── PreviewDialog             - 预览编辑弹窗组件
│   ├── editable-table        - 可编辑表格
│   ├── row-editor            - 行编辑器
│   └── validation-status     - 验证状态指示器
├── handleBatchImport()       - 执行批量导入
└── showResultDialog()        - 显示结果汇总
```

### 4.3 数据流

```
用户上传文件
    │
    ▼
parseExcelFile() → 原始数据数组
    │
    ▼
validateAllRows() → { validRows: [], invalidRows: [] }
    │
    ▼
显示预览弹窗（可编辑修正）
    │
    ▼
用户点击"开始导入"
    │
    ▼
逐条插入 common_medications 表
    │
    ▼
统计结果 → 显示汇总弹窗
```

## 5. 实现细节

### 5.1 Excel 模板生成

```typescript
function downloadTemplate() {
  const wb = XLSX.utils.book_new()

  // 工作表 1：填写说明
  const instructionData = [
    ['📋 填写说明'],
    ['• 标红单元格为必填项'],
    ['• 药品分类：从下拉列表选择...'],
    // ...
  ]
  const wsInstructions = XLSX.utils.aoa_to_sheet(instructionData)
  XLSX.utils.book_append_sheet(wb, wsInstructions, '填写说明')

  // 工作表 2：模板数据（含数据验证）
  const headers = ['药品名称', '通用名称', '药品分类', '生产厂家', '规格', '剂型', '外观描述', '剂量单位']
  const wsTemplate = XLSX.utils.aoa_to_sheet([headers, ...emptyRows])

  // 设置数据验证（下拉列表）
  wsTemplate['!dataValidations'] = {
    C2:C100: { type: 'list', formulae: ['降压药，降糖药，...'] },
    F2:F100: { type: 'list', formulae: ['tablet', 'capsule', 'liquid'] }
  }

  XLSX.utils.book_append_sheet(wb, wsTemplate, '模板数据')
  XLSX.writeFile(wb, `药品批量导入模板_${dateString}.xlsx`)
}
```

### 5.2 文件上传解析

```typescript
async function handleFileUpload(file: File) {
  const data = await file.arrayBuffer()
  const workbook = XLSX.read(data, { type: 'array' })
  const ws = workbook.Sheets[workbook.SheetNames[1]] // 模板数据工作表
  const rows = XLSX.utils.sheet_to_json(ws)

  const validationResults = rows.map((row, index) => ({
    rowNum: index + 2, // Excel 行号（从 1 开始，跳过表头）
    data: row,
    errors: validateRow(row)
  }))

  return validationResults
}
```

### 5.3 批量导入

```typescript
async function handleBatchImport(validRows: any[]) {
  const results = { success: 0, duplicate: 0, failed: 0, details: [] }

  for (const row of validRows) {
    const { error } = await supabase
      .from('common_medications')
      .insert([{ ...row, is_active: true }])

    if (error) {
      if (error.code === '23505') {
        results.duplicate++
        results.details.push({ name: row.name, reason: '已存在' })
      } else {
        results.failed++
        results.details.push({ name: row.name, reason: error.message })
      }
    } else {
      results.success++
    }
  }

  return results
}
```
