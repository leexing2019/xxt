## 1. 需求概述

### 1.1 背景
用户完成病史小结问答后，需要将生成的报告分享给他人（如家人、医生）。当前方案在 H5 端使用 html2canvas 工作正常，但在 App 端（uni-app + Capacitor）无法使用。

### 1.2 目标
- 在 App 端实现分享报告功能
- 生成精美设计图片（蓝色主题、专业医疗风格）
- 用户可以保存到相册或直接分享到微信/QQ 等社交平台

### 1.3 用户故事
> 作为用户，我希望将生成的病史报告以精美图片的形式分享给家人或医生，这样他们可以方便地查看我的健康状况。

## 2. 技术方案

### 2.1 架构决策

采用**平台差异化方案**：

| 平台 | 技术方案 | 原因 |
|------|----------|------|
| H5 | html2canvas | 已实现，工作良好 |
| App/小程序 | uni-app Canvas 2D API | 原生支持，跨平台一致 |

### 2.2 技术架构图

```
┌─────────────────────────────────────────────────────────────┐
│                     点击"分享报告"按钮                        │
└─────────────────────────┬───────────────────────────────────┘
                          │
                          ▼
              ┌───────────────────────┐
              │   平台判断 (条件编译)   │
              └───────────┬───────────┘
                          │
          ┌───────────────┼───────────────┐
          │               │               │
          ▼               ▼               ▼
    ┌──────────┐   ┌──────────┐   ┌──────────┐
    │   H5     │   │   App    │   │  小程序   │
    │html2canvas│   │  Canvas  │   │  Canvas  │
    └────┬─────┘   └────┬─────┘   └────┬─────┘

         └──────────────┼──────────────┘

              ┌───────────────────────┐
              │    统一预览弹窗        │
```

```
│  │                                   │
│  │  === 病史摘要 ===                 │
│  │                                   │
│  │  姓名：张三                       │
│  │  年龄：65 岁                       │
│  │  性别：男                         │
│  │  ...                              │
│  │                                   │
```

**颜色和字体**：

| 元素 | 属性 | 值 |
|------|------|-----|
| 顶部背景 | 渐变 | linear-gradient(135deg, #2196F3, #1976D2) |
| 标题文字 | 颜色/大小 | #FFFFFF / bold 24px |
| 日期文字 | 颜色/大小 | rgba(255,255,255,0.8) / 14px |
| 正文背景 | 颜色 | #FFFFFF |
| 正文字体 | 颜色/大小 | #333333 / 14px |
| 行高 | - | 1.8 |
| 底部背景 | 颜色 | #F8F9FA |
| 底部文字 | 颜色/大小 | #999999 / 12px |

## 4. 错误处理

| 错误场景 | 处理方式 |
|----------|----------|
| Canvas 创建失败 | Toast 提示"生成失败，请重试" |
| 绘制过程出错 | Toast + 降级方案 (提示截图) |
| 保存权限不足 | 引导用户开启存储权限 |
| 分享失败 | 降级为保存图片 |
